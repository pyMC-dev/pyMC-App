# pyMC-App

Public issue and bug tracker for the **pyMC App**.

## Why this repository exists

The main app repository is private/closed-source.
This repository is the public place for users and testers to:

- report bugs
- track known issues
- follow fixes and release-related discussion

## What to open here

Use this repository for:

- app crashes
- UI/UX issues
- platform-specific problems (Android, iOS, desktop)
- connectivity issues (BLE/TCP)
- regressions after updates

Please include as much detail as possible so issues can be reproduced quickly.

## Download

App builds are available at:

- https://files.pymc.dev

## How to report a bug

1. Open the **Issues** tab.
2. Choose **Bug Report**.
3. Fill out all required fields.

The template asks for key details such as:

- platform and OS version
- app version
- device/hardware
- connection type (BLE/TCP/offline)
- affected feature area
- steps to reproduce
- expected vs actual behavior
- screenshots/screen recordings (if helpful)

## Before opening a new issue

- search existing issues to avoid duplicates
- update to the latest available app version
- test again after restarting app/device (if relevant)

## Scope

This repository is for issue tracking only.

- source code is not published here
- pull requests are not expected for app source changes

## Security and sensitive reports

Please do **not** post secrets, private keys, personal data, or sensitive network details in public issues.

If you need to report a security-sensitive issue, open an issue with minimal detail and request a private contact channel.

## Thank you

Your reports help improve stability and reliability across all supported platforms.
